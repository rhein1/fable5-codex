# Fable-5 for Codex v0.3.0-alpha

This alpha release packages Fable-5 as an installable OpenAI Codex plugin for evidence-first audits, deep reviews, fact checks, codebase understanding, design options, and repo-wide sweeps.

## Highlights

- Six reusable Codex skills:
  - `$fable-audit`
  - `$fable-deep-review`
  - `$fable-fact-check`
  - `$fable-understand`
  - `$fable-design-options`
  - `$fable-sweep`
- Micro ECF-style run contracts and receipt fields.
- Large/high-risk task policy for real Codex subagents when the runtime exposes subagent tooling and the user has not opted out.
- Honest Workflow Trace reporting for multi-agent and single-agent multi-lens runs.
- Refreshed benchmark snapshot: `93.3 -> 98.7` average composite on the tiny workflow fixtures.
- npm installer path: `npx fable5-codex`.

## Install

```powershell
npx fable5-codex
```

GitHub fallback:

```powershell
npx github:rhein1/fable5-codex
```

Pinned Codex marketplace install:

```powershell
codex plugin marketplace add rhein1/fable5-codex --ref v0.3.0-alpha
codex plugin add fable5-codex@fable5-local
```

## Caveats

- This is alpha software.
- The public package includes Micro ECF-style contracts and reporting rules only.
- It does not include private Full ECF internals.
- Subagents are only real when Codex exposes a subagent tool and the final report records real subagent IDs or runtime-visible handles.
