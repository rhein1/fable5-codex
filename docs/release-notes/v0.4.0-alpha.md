# Fable-5 for Codex v0.4.0-alpha

Fable-5 v0.4 adds a first-class GPT-5.6 Sol Ultra profile for evidence-first multi-agent codebase work.

## Highlights

- `gpt-5.6-sol` plus `ultra` is now the default in the packaged PowerShell, Bash, and benchmark runners.
- A ready-to-copy Codex config keeps four-agent Ultra fan-out bounded with `agents.max_depth = 1`.
- Plugin metadata, install docs, and branded imagery now explain the Sol Ultra profile.
- Fable's existing ECF contracts, authority split, independent lenses, verification, and Workflow Trace remain the governance layer around real Codex subagents.
- A new matched Sol Ultra benchmark publishes all six outputs plus quality and latency charts. Average composite improved `81.7 -> 100.0`; average wall time increased `144.5s -> 344.0s`.

The benchmark deliberately disabled subagents on its tiny fixtures, so it measures Fable workflow discipline rather than multi-agent gains. One provider-capacity failure was retried in place with the same configuration and is disclosed in the run record.

## Install

```powershell
npx github:rhein1/fable5-codex
```

Then select GPT-5.6 Sol and Ultra in Codex, or use `plugins/fable5-codex/templates/sol-ultra.config.toml`.
