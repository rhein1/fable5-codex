# Fable-5 for Codex v0.4.0-alpha

Fable-5 v0.4 adds a first-class GPT-5.6 Sol Ultra profile for evidence-first multi-agent codebase work.

## Highlights

- `gpt-5.6-sol` plus `ultra` is now the default in the packaged PowerShell, Bash, and benchmark runners.
- A ready-to-copy Codex config bounds Ultra delegation at `agents.max_threads = 6` and `agents.max_depth = 1`; the four-lens Fable audit is a workflow example, not Codex's default worker count.
- Plugin metadata, install docs, and branded imagery now explain the Sol Ultra profile.
- Fable's existing ECF contracts, authority split, independent lenses, verification, and Workflow Trace remain the governance layer around real Codex subagents.
- A matched Sol Ultra benchmark publishes all six outputs plus lexical workflow-format and latency charts. Average rubric composite improved `81.7 -> 100.0`; average wall time increased `144.5s -> 344.0s`.

Retrospective qualification: the benchmark deliberately disabled subagents on its tiny fixtures, used one trial per arm, and scored lexical/structural signals rather than semantic quality. Its pre-alpha.3 plugin arm was not cleanly isolated from the active user environment, so the result is historical workflow-format evidence rather than plugin-only causal proof. One provider-capacity failure was retried in place with the same configuration and is disclosed in the run record.

## Install

```powershell
npx github:rhein1/fable5-codex#v0.4.0-alpha
```

Then select GPT-5.6 Sol and Ultra in Codex, or use `plugins/fable5-codex/templates/sol-ultra.config.toml`.
